# My Todo App

これは、React + TypeScript + Tailwind CSS を使って開発したシンプルなTodoアプリです。タスクの追加や完了状態の切り替えができる機能を備えています。

## 📦 使用技術

- React 18
- TypeScript
- Vite
- Tailwind CSS

## 🔧 主な機能

- Todoの追加
- 完了状態の切り替え
- 状態管理（useState）
- コンポーネント分割（Input / List / Item）

## 💻 開発環境構築

```bash
git clone https://github.com/Kirin1013/my-todo-app.git
cd my-todo-app
npm install
npm run dev
